# 🎵 YT Music Playlist Enhancer

[![Version](https://img.shields.io/badge/version-1.1.0-00f0ff.svg?style=flat-square)](https://github.com/AexZeM/yt-music-playlist-enhancer)
[![Platform](https://img.shields.io/badge/platform-Chrome-red.svg?style=flat-square)](https://music.youtube.com )
[![License](https://img.shields.io/badge/license-MIT-lightgrey.svg?style=flat-square)](LICENSE)

YouTube Music has a really substantial music library that I love. However, there were a few things that felt off. As my playlists grew longer and longer, 
I started having issues: I couldn't find specific covers or songs quickly, and it became nearly impossible to manage large playlists.
I searched for an extension or software to solve this but none of them satisfied me. So I decided to develop my own extension to fix these problems. 
I've done small projects before but I never shared them soo this is my first big project and the first one I'm sharing with the world.
I hope it helps others who are experiencing the same frustrations with YTM.

---

### Key Features

* **Tag System** — Categorize and filter your music with a custom tagging system
* **Search Bar** — Quickly find specific songs within any playlist
* **Duplicate Scanner** — Identify and manage multiple instances of the same song
* **Export System** — Save your data in `.md`, `.csv`, or `.json` formats

### Manual Installation

1.  **Download** the repository as a ZIP file and extract it
2.  Open your browser's **Extension Settings** (`chrome://extensions`)
3.  Enable **Developer Mode** (toggle in the top right)
4.  Click **Load unpacked** and select the extracted folder
